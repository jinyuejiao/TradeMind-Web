TradeMind Web - 浏览器内Web服务器

注意：这是一个在浏览器中运行的Web服务器模拟器，用于访问TradeMind Web的所有页面。

使用说明

如何使用本服务器：

  1. 保持此页面打开 - 这是您的服务器控制中心
  2. 点击下方的链接 - 直接在浏览器中打开各个页面
  3. 使用浏览器的前进/后退按钮 - 在页面之间导航
  4. 刷新页面 - 重新加载当前页面

优势：无需安装任何软件，无需启动外部服务器，直接在浏览器中运行。

可用页面

核心功能页面

  • 登录页面 (login.html)
  • 注册页面 (register.html)
  • 主页面/工作台 (index.html)

业务模块页面

  • 客户CRM (customer-manage.html)
  • 产研中心 (production.html)
  • 供应链管理 (supplier.html)
  • 订单管理 (order-manage.html)

技术说明

为什么使用浏览器内服务器？

由于系统环境的限制，无法启动传统的Web服务器。通过直接在浏览器中打开HTML文件，您可以：

  • 查看所有页面的完整布局和样式
  • 测试页面的基本功能
  • 浏览所有业务模块的内容
  • 无需安装任何额外软件

注意：由于是直接打开文件，一些需要服务器支持的功能（如API调用、跨域请求）可能无法正常工作，但页面的基本展示和交互功能都可以正常使用。
